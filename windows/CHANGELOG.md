## 1.1.2 — 2026-07-16 — 栋哥浅色主题

### 新增

- Windows 首页接入与 macOS 相同的“心理问题白板”横幅
- 专属文案：`栋哥的 dbskill`、`一个开源分身`、`心理问题`、`试试就知道`

### 改进

- 从旧的固定人物与粉色装饰切换为暖白、炭色和克制砖红配色
- 横幅使用标准 `cover` 裁切，移除只适用于旧图片的拍立得与超倍率背景裁切
- 重复注入时刷新背景对象与主题装饰，避免继续显示旧图片或旧文案
- 删除不再使用的旧固定人物背景，避免打包时混入旧主题素材

### 说明

- 仍通过本机回环 CDP 注入，不修改 WindowsApps 或官方 Codex 文件
