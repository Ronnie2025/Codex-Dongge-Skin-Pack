# Dongge Codex Theme Pack

An independent Dongge theme pack for the Codex desktop app on macOS and Windows.

[Download the latest package](https://github.com/Ronnie2025/Codex-Dongge-Skin-Pack/releases/latest)

The installer lets users choose among four bundled images: Language in Use, Open-source Workbench, Psychology Placard, and Classic Whiteboard. The installed theme switcher exposes the same four choices.

## Safety

- Does not modify the official Codex bundle, `app.asar`, or `WindowsApps`.
- Does not automatically quit or restart Codex.
- Does not use `--restart-existing`.
- Does not install LaunchAgents, timers, or recurring auto-reapply jobs.
- If Codex is already running without the required debug port, the launcher stops and asks the user to quit it manually.

The theme uses CDP on the loopback interface. Do not run untrusted local software while it is active.
