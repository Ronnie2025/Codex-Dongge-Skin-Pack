# 栋哥浅色皮肤设计简报

## 项目范围

- 使用范围：团队内部，可上传 GitHub。
- 交付范围：外观 + 专属文案。
- 平台：macOS 与 Windows。
- 当前阶段：候选 01 已定稿，并完成 macOS 与 Windows 接入。
- 人物呈现：尽量保留真实摄影感，不把人物做成明显的 AI 形象。

## 人物核心

- 口头禅：`心理问题`、`试试就知道`。
- 代表项目：`dbskill`，栋哥开源的个人分身 Skill。
- 思想偏好：维特根斯坦，关注问题、语言、用法与情境。
- 本轮不以“商业诊断”或“真实行动”为主要人物标签。

## 视觉原则

- 浅色、克制、真实、带编辑摄影或文化杂志质感。
- 人物位于画面中右侧，左侧至少 45% 保持低细节留白，供 Codex 原生标题与控件使用。
- 保留短黑发、细框眼镜、自然脸型和真实肤质；不过度磨皮，不重塑五官。
- `心理问题`、`试试就知道`、`dbskill` 通过白板、举牌、纸张或开源手册等真实物件出现。
- 维特根斯坦元素通过语言、用法、情境、问题等关系表达，不使用哲学家肖像、雕像或伪造名言。
- 不在壁纸中绘制 Codex 控件、输入框、按钮或其他假 UI。

## 禁止方向

- 微商风、成功学、商业导师或精英顾问包装。
- low、红黄促销、廉价霓虹、赛博 HUD、AI 机器人与全息数字分身。
- 短视频封面、粉色抠图描边、气泡字、表情包与密集大字报。
- 塑料皮肤、夸张景深、过度电影光、假笑和明显 AI 五官漂移。

## 候选方向

1. 心理问题白板：真实讲解现场，白板呈现口头禅与问题关系。
2. 心理问题举牌：复用人物最强辨识动作，但改为克制的编辑摄影。
3. 维特根斯坦案头：语言、用法、情境与问题的纸面痕迹。
4. dbskill 开源工作台：用装订手册、技能卡和版本贴纸表达“开源分身”。
5. 试试就知道：日常讨论场景，用简单纸样表达先试再看。

## 选择结论

- 已选：候选 01「心理问题白板」。
- 正式母版：`background.png`。
- 选择原因：人物一致性最好，`心理问题`、`试试就知道`、`dbskill` 三个核心信息均自然进入真实讲解现场，且左侧留白足够承载 Codex 原生界面。
- 专属文案：`栋哥的 dbskill` / `一个开源分身` / `心理问题` / `先看看是什么问题` / `试试就知道`。
