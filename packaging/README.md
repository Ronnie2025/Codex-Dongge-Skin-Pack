# 栋哥 Codex 主题资源包

这个压缩包支持 macOS 与 Windows，内置四套栋哥主题：

1. 语言的用法
2. 开源工作台
3. 心理问题举牌
4. 经典白板

安装时会询问默认使用哪张图片，安装后也能从“选择风格”入口重新选择。

## macOS

进入 `macOS` 文件夹，双击 `Install Codex Dream Skin.command`。安装完成后，从生成的 `栋哥 Codex.app` 打开。

## Windows

先安装 Node.js，进入 `Windows` 文件夹，再双击 `Install Codex Dream Skin.cmd`。安装后从 `栋哥 Codex` 快捷方式打开。

## 重要说明

- 工具不会自动退出或重启 Codex。
- 工具不使用 `--restart-existing`，也不安装后台定时任务。
- 如果官方 Codex 已经打开，请手动退出后再打开 `栋哥 Codex`。
- `Restore` / 恢复入口可以停用皮肤并恢复官方外观。
- 不修改官方 Codex 安装包、`app.asar` 或 `WindowsApps`。
